# 🔐 密室逃脱 - 游戏设计文档 (GDD)

## 1. 概述

| 项目 | 内容 |
|------|------|
| 游戏名称 | 密室逃脱 (Escape Room) |
| 游戏类型 | 第一人称解谜 / 密室逃脱 |
| 平台 | Web Browser (PC) |
| 引擎 | Three.js (WebGL) |
| 开发周期 | 单文件架构 |
| 目标玩家 | 喜欢解谜、推理的玩家 |

## 2. 游戏概述

玩家在一间陌生的昏暗房间中醒来，门被锁住。必须通过探索环境、收集线索、解开谜题来找到逃出密室的方法。

### 2.1 核心体验
- **探索感**：昏暗环境 + 手电筒，营造紧张氛围
- **发现感**：隐藏的线索藏在日常物品中
- **成就感**：环环相扣的谜题链，每解开一步都有满足感
- **沉浸感**：第一人称视角 + 音效 + 光影

## 3. 游戏机制

### 3.1 移动系统
- WASD 移动 + 鼠标环顾（Pointer Lock）
- Shift 冲刺
- 碰撞检测（墙壁阻挡）

### 3.2 互动系统
- 视线检测（Raycasting）
- 5 米互动范围
- E 键互动
- 物体高亮提示

### 3.3 物品系统
- 6 格物品栏
- 点击选择物品
- 物品用于解锁/触发机关

### 3.4 笔记系统
- 自动记录发现的线索
- 显示在右上角
- 帮助玩家回顾信息

### 3.5 提示系统
- 8 条逐步提示
- 根据进度解锁
- 卡住时提供引导

## 5. 谜题设计

### 5.1 谜题链路
```
红色旧书 → 纸条("时间是答案")
    ↓
挂钟(3:15) → 密码提示
    ↓
油画(移动) → 暗格 → 抽屉钥匙
    ↓
抽屉(解锁) → 照片("密码藏在能看到时间的地方")
    ↓
衣柜 → 纸条("灯下有秘密")
    ↓
台灯(打开) → 灯座纸条("门的密码是万能钥匙")
    ↓
保险箱(密码3115) → 万能钥匙
    ↓
门(解锁) → 逃脱成功！
```

### 5.2 谜题详细设计

#### 谜题 1：红色旧书
- **位置**：书架第二层
- **触发**：靠近 + E 键
- **内容**：纸条"时间是答案。看看墙上的钟。"
- **设计意图**：引导玩家注意挂钟

#### 谜题 2：挂钟
- **位置**：后墙中央
- **触发**：靠近 + E 键
- **内容**：指针停在 3:15
- **设计意图**：提供保险箱密码

#### 谜题 3：油画
- **位置**：左墙
- **触发**：靠近 + E 键
- **内容**：移动油画，发现暗格，获得抽屉钥匙
- **设计意图**：物品获取 + 引导下一步

#### 谜题 4：抽屉
- **位置**：书桌右侧
- **触发**：使用抽屉钥匙 + E 键
- **内容**：照片背面"保险箱密码藏在能看到时间的地方"
- **设计意图**：强化密码与时间的关联

#### 谜题 5：衣柜
- **位置**：左后方
- **触发**：靠近 + E 键
- **内容**：纸条"灯下有秘密"
- **设计意图**：引导玩家使用台灯

#### 谜题 6：台灯
- **位置**：书桌上
- **触发**：靠近 + E 键
- **内容**：打开灯，灯座下纸条"门的密码是万能钥匙"
- **设计意图**：暗示最终目标

#### 谜题 7：保险箱
- **位置**：书架旁
- **触发**：输入密码 3115
- **内容**：获得万能钥匙
- **设计意图**：核心解谜环节

#### 谜题 8：门
- **位置**：后墙
- **触发**：使用万能钥匙
- **内容**：逃脱成功
- **设计意图**：游戏结束

## 6. 音效设计

| 音效 | 触发 | 实现 |
|------|------|------|
| 点击音 | 任何互动 | WebAudio 800Hz 正弦波 |
| 解锁成功 | 正确操作 | 上行三连音 |
| 错误音 | 错误操作 | 低频锯齿波 |
| 开门音 | 打开门 | 低频噪音 |
| 脚步声 | 移动 | 随机低频短音 |

## 7. 视觉设计

### 7.1 色调
- 主色调：深棕、暗红、暗蓝
- 高亮色：橙红色 (#ff8866) 用于 UI 和互动提示
- 光源色：暖黄 (吊灯)、冷白 (手电筒)

### 7.2 纹理
- 全部程序化生成（Canvas 2D）
- 木地板：随机色调木纹
- 墙面：条纹壁纸 + 老化污渍
- 书籍：多种颜色随机生成

### 7.3 后期效果
- ACES 色调映射
- 柔和阴影 (PCFSoftShadowMap)
- 雾气效果
- 闪烁灯光
