# 🔐 密室逃脱 - Escape Room

一个基于 Three.js 的全 3D 第一人称密室逃脱游戏。在昏暗的房间里寻找线索、解开谜题，逃出密室！

## 🎮 游戏特色

### 🏠 写实密室环境
- **完整房间** — 木地板纹理、老化墙面、砖砌封死的窗户
- **精细家具** — 书架（带大量书籍）、书桌（带抽屉）、衣柜、床铺、挂钟
- **氛围灯光** — 闪烁的吊灯、可开关的台灯、手电筒
- **细节物件** — 杯子、纸张、地毯、油画、老式挂钟（带钟摆）

### 🧩 谜题设计
游戏包含一套连贯的谜题链：
1. 🔍 探索房间，找到**红色旧书**中的线索
2. 🕐 观察**挂钟**上的时间（3:15）
3. 🖼️ 移动**油画**发现暗格，获得抽屉钥匙
4. 📦 用钥匙打开**抽屉**，获得更多线索
5. 🚪 打开**衣柜**，找到灯下秘密的提示
6. 💡 打开**台灯**，发现灯座下的纸条
7. 🔒 用密码 **3115** 打开**保险箱**
8. 🔑 获得**万能钥匙**，打开房门逃脱！

### 🎯 操作方式
| 操作 | 功能 |
|------|------|
| `WASD` | 移动 |
| `鼠标` | 环顾四周 |
| `E` | 与物体互动 |
| `Shift` | 冲刺 |
| `鼠标点击` | 锁定鼠标（FPS模式） |
| `💡` | 提示按钮 |

### 📝 笔记系统
- 自动记录发现的线索
- 物品栏（最多6格）
- 计时器统计逃脱用时
- 提示系统（逐步引导）

## 🚀 运行方式

直接在浏览器中打开 `index.html` 即可。

```bash
# 或使用本地服务器
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 📦 技术栈

- **Three.js** — 3D 渲染引擎
- **Web Audio API** — 环境音效（开门声、解锁声、脚步声等）
- **Canvas 2D** — 程序化纹理生成
- **Pointer Lock API** — 第一人称视角控制
- 纯原生 JavaScript，零依赖
